# 灵感画布 - AI 工作流可视化编排系统

## 概述

灵感画布是一个全新的可视化 AI 工作流编排系统，允许你通过拖拽节点的方式组合系统的各种接口，创建自动化工作流。

## 核心功能

### 1. 节点库
左侧面板包含所有可用的节点，分为四大类：

#### 🎯 触发器
- **手动触发**：手动启动工作流
- **定时触发**：按计划自动执行（支持 cron 表达式）

#### ⚡ 动作
- **AI 对话**：调用 AI 模型生成回复
- **生成图片**：使用 AI 生成图片
- **发送消息**：发送消息给指定用户
- **HTTP 请求**：发送 HTTP 请求到外部 API
- **创建笔记**：保存内容到笔记模块

#### 🔀 逻辑
- **条件判断**：根据条件分支执行
- **循环**：遍历数组或重复执行
- **延迟**：等待指定时间

#### 📊 数据
- **数据转换**：转换或处理数据

### 2. 画布操作

#### 添加节点
- 点击左侧节点库中的任意节点，自动添加到画布中心
- 或者在空画布点击"添加第一个节点"按钮

#### 移动节点
- 鼠标按住节点拖动即可移动位置
- 节点会自动保存位置

#### 删除节点
- 点击节点右上角的 × 按钮删除节点
- 删除节点会同时删除相关连接

#### 画布导航
- **平移**：鼠标在空白区域拖动
- **缩放**：使用工具栏的 + / - 按钮
- 当前缩放比例显示在工具栏中间

### 3. 工作流示例

#### 示例 1：AI 图片生成 + 发送消息
```
手动触发 → AI 对话（生成图片描述） → 生成图片 → 发送消息
```

#### 示例 2：定时任务 + 条件判断
```
定时触发 → HTTP 请求（获取数据） → 条件判断 → 发送消息 / 创建笔记
```

#### 示例 3：循环处理
```
手动触发 → 循环 → AI 对话 → 数据转换 → 创建笔记
```

### 4. 保存到秩序

完成工作流设计后，点击右上角"保存到秩序"按钮，工作流会自动保存到秩序模块，可以在那里执行和管理。

## 节点详解

### 输入输出端口
每个节点都有输入端口（左侧）和输出端口（右侧）：
- **输入端口**：接收上游节点的数据
- **输出端口**：传递数据给下游节点

### 节点配置
选中节点后可以配置节点参数（未来版本将添加配置面板）

## 快捷键

- `Esc`：取消选择
- 鼠标滚轮：缩放画布（未来版本）
- `Space + 拖动`：平移画布（未来版本）

## 与秩序模块的关系

- **灵感画布**：可视化设计工作流，快速原型
- **秩序模块**：执行和管理工作流，生产环境

两者互补，画布专注于创意和设计，秩序专注于执行和管理。
