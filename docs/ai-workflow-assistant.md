# AI 工作流助手使用指南

## 概述

AI 工作流助手是灵感画布的智能功能，它可以理解你的需求，自动设计并生成工作流节点。这是一个真正的 AI Agent，能够：

1. **理解自然语言需求**：用中文描述你想做什么
2. **智能节点组合**：根据可用节点自动设计最优工作流
3. **一键生成**：直接在画布上创建节点，无需手动拖拽

## 如何使用

### 1. 打开 AI 助手

点击工具栏的 **"显示 AI 助手"** 按钮，左侧会出现 AI 助手面板。

### 2. 描述你的需求

在输入框中用自然语言描述你想要的工作流，例如：

- "帮我做一个每天早上 9 点自动发送天气预报的工作流"
- "创建一个工作流：用 AI 生成图片描述，然后生成图片，最后发送给我"
- "设计一个工作流：定时获取新闻，用 AI 总结，保存到笔记"
- "做一个循环处理任务列表的工作流"

### 3. AI 自动生成

AI 会：
1. 分析你的需求
2. 选择合适的节点
3. 设计节点流程
4. 返回工作流说明和 JSON 定义
5. 自动在画布上创建节点

### 4. 调整和保存

生成后你可以：
- 拖动节点调整位置
- 添加或删除节点
- 点击"保存到秩序"保存工作流

## 快速开始示例

### 示例 1：AI 图片生成工作流

**需求**：
```
创建一个工作流：用 AI 生成图片描述，然后生成图片，最后发送给我
```

**AI 会生成**：
```
手动触发 → AI 对话（生成图片描述） → 生成图片 → 发送消息
```

### 示例 2：定时新闻摘要

```

```

**AI 会生成**：
```
定时触发 → HTTP 请求（获取新闻） → AI 对话（总结） → 创建笔记
```

### 示例 3：批量处理任务

**需求**：
```
做一个循环处理任务列表的工作流
```

**AI 会生成**：
```
手动触发 → 循环 → AI 对话（处理单个任务） → 数据转换 → 创建笔记
```

## AI 助手的工作原理

### 1. 节点知识库

AI 助手了解所有可用节点：

- **触发器**：手动触发、定时触发
- **AI 动作**：AI 对话、生成图片
- **消息动作**：发送消息
- **HTTP 请求**：调用外部 API
- **逻辑控制**：条件判断、循环、延迟
- **数据处理**：数据转换、创建笔记

### 2. 设计原则

AI 遵循以下原则设计工作流：

1. **必须从触发器开始**：每个工作流都需要一个触发器
2. **逻辑连贯**：节点之间的输入输出要匹配
3. **简洁高效**：避免不必要的节点
4. **考虑边界情况**：包含错误处理和条件判断

### 3. 输出格式

AI 会返回：
- **工作流说明**：用中文解释工作流的逻辑
- **JSON 定义**：包含所有节点的配置
- **自动创建**：直接在画布上生成节点

## 高级技巧

### 1. 明确需求

越具体的需求，AI 生成的工作流越准确：

❌ 不好：
```
做一个工作流
```

✅ 好：
```
创建一个工作流：每天早上 9 点，用 AI 生成一句励志名言，发送给我
```

### 2. 分步描述

复杂需求可以分步描述：

```
我想做一个工作流：
1. 定时触发（每小时）
2. 调用天气 API 获取数据
3. 如果温度超过 30 度，发送高温提醒
4. 否则，什么都不做
```

### 3. 指定细节

可以指定具体的配置：

```
创建一个 AI 对话节点，prompt 是"生成一个科技新闻标题"，
然后把结果保存到笔记，标题是"今日科技"
```

### 4. 迭代优化

生成后可以继续对话优化：

```
用户：帮我做一个图片生成工作流
AI：[生成基础工作流]
用户：在生成图片之前，先用 AI 优化一下 prompt
AI：[更新工作流，添加 AI 对话节点]
```

## 注意事项

### 1. 需要配置 AI 模型

使用前请确保：
- 在"接口中枢"配置了 AI 模型
- 选择了当前激活的模型配置

### 2. 生成的是节点，不是连接线

当前版本 AI 只生成节点，节点之间的连接线需要手动创建（未来版本会支持）。

### 3. 节点配置可能需要调整

AI 生成的节点配置是默认值，你可能需要：
- 调整 AI 的 prompt
- 修改 HTTP 请求的 URL
- 设置定时触发的时间

### 4. 复杂工作流建议分步生成

对于非常复杂的工作流，建议：
1. 先让 AI 生成主要流程
2. 再手动添加细节节点
3. 或者分多次对话逐步完善

## 与传统方式对比

### 传统方式
1. 浏览节点库
2. 逐个拖拽节点
3. 手动连接
4. 配置参数

⏱️ 耗时：5-10 分钟

### AI 助手方式
1. 描述需求
2. AI 自动生成
3. 微调（可选）

⏱️ 耗时：30 秒

## 常见问题

### Q: AI 生成的工作流不符合预期怎么办？

A: 可以：
1. 继续对话，告诉 AI 哪里需要调整
2. 手动修改节点
3. 重新描述需求，让 AI 重新生成

### Q: AI 助手支持哪些语言？

A: 目前主要支持中文，也可以理解简单的英文需求。

### Q: 生成的节点可以删除吗？

A: 可以，点击节点右上角的 × 按钮删除。

### Q: AI 会保存我的对话历史吗？

A: 当前会话的对话会保留，但刷新页面后会清空。
