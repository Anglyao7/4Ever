# 交耳模块 - 添加菜单功能与弹窗

## 功能概述

在交耳（聊天）模块的搜索框右侧添加了一个"+"按钮，点击后会弹出一个下拉菜单，包含以下三个选项，每个选项都会打开对应的弹窗进行后续操作：

### 1. 添加好友
- 点击后打开添加好友弹窗
- 提供搜索框搜索用户（用户名/邮箱/昵称）
- 显示搜索结果列表
- 可直接点击"加好友"按钮添加
- 显示好友状态（已是好友/已申请/加好友）

### 2. 新建群组
- 点击后打开新建群组弹窗
- 输入群组名称
- 从好友列表中选择成员
- 支持搜索过滤好友
- 已选中的成员会高亮显示
- 点击"创建"按钮完成群组创建

### 3. 新建AI
- 点击后打开新建AI弹窗
- 输入AI名字
- 设置AI性格描述
- 点击"创建"按钮完成AI创建并切换到AI会话模式

## 实现细节

### 组件修改

**文件**: `frontend/src/ChatPanel.tsx`

#### 新增类型
- `DialogType`: `"add-friend" | "new-group" | "new-ai" | null`

#### 新增状态
- `activeDialog`: 当前激活的弹窗类型
- `dialogSearchQuery`: 弹窗内的搜索关键词
- `dialogSearchResults`: 弹窗内的搜索结果
- `dialogSearchLoading`: 弹窗搜索加载状态
- `newGroupName`: 新建群组的名称
- `newGroupMembers`: 新建群组选中的成员ID列表
- `newAiName`: 新建AI的名字
- `newAiPersona`: 新建AI的性格描述

#### 新增函数
- `closeDialog()`: 关闭弹窗并重置所有弹窗状态
- `handleCreateGroup()`: 创建群组（待实现API调用）
- `handleCreateAi()`: 创建AI并切换到AI模式
- `toggleGroupMember()`: 切换群组成员选中状态

#### 新增组件
- `AddFriendDialog`: 添加好友弹窗组件
- `NewGroupDialog`: 新建群组弹窗组件
- `NewAiDialog`: 新建AI弹窗组件

### 交互逻辑

#### 菜单交互
1. 点击"+"按钮切换菜单显示状态
2. 点击菜单外部或按ESC键自动关闭菜单
3. 选择菜单项后打开对应弹窗

#### 弹窗交互
1. **模态弹窗**: 打开时锁定背景滚动，点击背景或按ESC键关闭
2. **搜索功能**: 
   - 添加好友弹窗：实时搜索用户
   - 新建群组弹窗：从好友列表搜索过滤
3. **验证逻辑**:
   - 添加好友：无需验证（搜索即可）
   - 新建群组：群组名称必填
   - 新建AI：AI名字必填
4. **自动关闭**: 创建成功或取消后自动关闭弹窗

### 样式设计

#### 下拉菜单
- 位置：搜索框与"+"按钮水平排列
- 按钮：38x38px，圆角8px
- 菜单：右对齐，最小宽度160px
- 动画：淡入效果（150ms）

#### 弹窗样式
- **布局**: 
  - 居中显示，最大宽度520px
  - 最大高度85vh，内容区域可滚动
- **结构**:
  - Header: 标题、描述、关闭按钮
  - Body: 表单内容、搜索、列表等
  - Footer: 取消和确定按钮
- **动画**: 
  - 背景淡入（200ms）
  - 弹窗缩放+平移（250ms）
- **配色**: 完全支持深色模式

#### 表单元素
- 输入框：10px内边距，圆角8px
- 文本域：最小高度100px，可垂直调整
- 成员列表：可滚动，最大高度300px
- 选中状态：背景色+边框色高亮

## 技术亮点

1. **独立搜索上下文**: 弹窗内的搜索不影响主界面搜索
2. **防抖搜索**: 220ms防抖避免频繁请求
3. **焦点管理**: 弹窗打开后自动聚焦到主要输入框
4. **滚动锁定**: 弹窗打开时锁定背景滚动
5. **状态隔离**: 关闭弹窗时自动重置所有相关状态
6. **类型安全**: 使用TypeScript联合类型管理弹窗状态
